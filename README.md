# DailyNews-master
个人练习的项目

>天天日报

天天日报是一款结合天气预报的资讯类应用，首页有热点新闻和新闻列表，热点新闻使用轮播图显示，下方是新闻列表，点击任意新闻可以进入到新闻详情页。在首页的右上角有分享功能，可以分享到手机里其他的应用。

侧滑菜单栏有今天、明后天的天气预报情况，也可以点击按钮刷新天气预报情况，下方还有返回首页、关于项目、问题反馈和退出应用四个功能。
